سياسة الخصوصية لتطبيق Mutaz Code

آخر تحديث: 22 سبتمبر 2025

1. مقدمة

مرحبًا بك في تطبيق Mutaz Code! نحن ملتزمون بحماية خصوصية مستخدمينا. هذه السياسة توضح كيفية جمعنا واستخدامنا للبيانات، إن وجدت، عبر تطبيقنا.

2. جمع البيانات

تطبيق Mutaz Code يعتمد على WebView لعرض الموقع https://mutazali12.github.io/schoolm/. نحن لا نجمع بيانات شخصية مباشرة من المستخدمين. ومع ذلك، قد يقوم الموقع أو الخدمات الخارجية المدمجة فيه (مثل الإعلانات أو أدوات التحليل) بجمع بيانات مثل عنوان IP أو ملفات تعريف الارتباط. لمزيد من التفاصيل، يرجى الرجوع إلى سياسة الخصوصية للموقع المذكور.

3. استخدام البيانات

لا نستخدم أي بيانات شخصية لأغراض تجارية. أي بيانات قد تُجمع تكون تحت سيطرة مزودي الخدمات الخارجية، وننصح بمراجعة سياساتهم.

4. حماية بيانات الأطفال

تطبيق Mutaz Code لا يستهدف الأطفال دون سن 13 عامًا. إذا كنت تعتقد أن طفلًا قدم بياناته، يرجى التواصل معنا فورًا لإزالتها.

5. التواصل معنا

إذا كانت لديك أي استفسارات حول سياسة الخصوصية، يرجى مراسلتنا عبر البريد الإلكتروني: mutazlifezone@gmail.com (يرجى استبدال العنوان ببريدك الفعلي).

6. التغييرات في السياسة
